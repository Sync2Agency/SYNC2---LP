## 3. SEO Técnico (Checklist de Implementação)

- [x] **Schema.org:** Implementado Organization e LocalBusiness no `App.tsx`.
- [x] **Mobile-First:** Design otimizado para dispositivos móveis com Tailwind.
- [x] **Core Web Vitals:** Otimização de imagens (Picsum com blur) e carregamento assíncrono.
- [x] **SSL/HTTPS:** Garantido pela infraestrutura de deployment.
- [x] **Meta Tags:** Dinâmicas implementadas via React Helmet.
- [ ] **Sitemap.xml:** Sugerido a criação via script de build.
- [ ] **Google Search Console:** Recomenda-se integrar após o deploy do domínio.

## 4. Estratégia de Backlinks e Autoridade

1.  **Parcerias com Influenciadores B2B:** Guest posts em blogs de tecnologia e negócios no Japão.
2.  **Diretórios Locais:** Cadastro em diretórios corporativos (ITreview.jp, etc).
3.  **PR Digital:** Divulgação de resultados de pesquisas trimestrais sobre o mercado de SNS no Japão (Link Building Orgânico).
4.  **LinkedIn Expertise:** Atração de tráfego qualificado e links via posts de autoridade do CEO.
